# Contributing to Trading Journal

Thank you for your interest in improving the Trading Journal application! 

## Ways to Contribute
1. **Reporting Bugs:** Create an issue describing the bug, how to reproduce it, and the environment.
2. **Suggesting Enhancements:** Create an issue outlining your proposed new feature and why it would be beneficial (e.g., adding integration for newer prop firm analytics).
3. **Pull Requests:** 
   - Fork the repo and create your branch from `main`.
   - Ensure the code adheres to our current React (Vite/Tailwind v4) standards.
   - Test your changes using the local SQLite db and `npm run dev`.
   - Issue a pull request with a descriptive summary of your changes.

## Development Setup
Please refer to the setup instructions outlined in the primary `README.md` file at the root of the project to initialize the Node.js backend and React frontend environments.

## Code of Conduct
Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms—be respectful and collaborative!
